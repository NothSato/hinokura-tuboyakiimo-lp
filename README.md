# つぼ焼き芋 火乃蔵 - 記事型ランディングページ

いわき市の焼き芋専門店「つぼ焼き芋 火乃蔵」の記事型ランディングページです。

## 店舗情報

- **店名**: つぼ焼き芋 火乃蔵（ひのくら）
- **特徴**: 糖度16度の紅はるかを2時間に1回丁寧に焼き上げ
- **価格**: 486円（税込）
- **食感**: "ねっとり"とした極上の仕上がり
